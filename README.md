O Recicla Eletrônico Paulista (REP) é uma plataforma que facilita o descarte correto de resíduos eletrônicos no estado de São Paulo, conectando usuários a pontos de coleta e receptores.
Desenvolvido com base no Design Thinking, o projeto utiliza tecnologias como, JavaScript, HTML e CSS para criar um sistema intuitivo e seguro.
Com uma abordagem sustentável, o REP busca reduzir os impactos ambientais do descarte inadequado, promover a conscientização da população e estimular a reciclagem.
Espera-se que a iniciativa gere um impacto positivo ao aprimorar a gestão de resíduos eletrônicos e contribuir para um meio ambiente mais limpo.
